## Galactic Squad - Gaming Website
A modern, responsive gaming website featuring a futuristic galactic theme with stunning visual effects and interactive elements using **HTML** , **CSS** , **JavaScript**.

---


### Features
1. **Futuristic Theme**:
 Dark space-inspired design with gradient backgrounds and glowing effects

2. **Modern Typography:** Uses Orbitron and Roboto fonts for a sci-fi aesthetic
 
3. **Responsive Layout:**
    Optimized for desktop, tablet, and mobile devices
   
4. **Smooth Animations:**
 Floating hero image, scroll-triggered animations, and hover effects

5. **Interactive Elements:**
   Animated navigation, back-to-top button, and dynamic team cards

6. There is a Contact form section where you can ask your doubt/feedback 

7. Social media links and copyright information

## Installation & Setup

- Clone or download the repository
- Ensure all image assets are in the root directory
- Configure EmailJS credentials in script.js
- Open index.html in a web browser
- For development, use a local server for best results

Clone the Repository:
```sh
git clone https://github.com/AkshatKardak/Game-Website.git
cd Game-Website
```
